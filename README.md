### 🎭 Blob Interactivo Avanzado
El sitio cuenta con un **sistema de blob dinámico** completamente personalizable:

#### 🔧 Características Técnicas
- **Sistema CSS/JS puro** - Sin dependencias SVG, completamente flexible
- **Morphing orgánico** - 6 formas asimétricas que se transforman suavemente
- **3 capas visuales** - Fondo colorido + imagen recortada + borde superior
- **9 imágenes de perfil** - Rotación automática cada 4 segundos
- **Interacción del mouse** - Movimiento reactivo y cambios aleatorios de imagen

#### 🎮 Controles Dinámicos
- **Atajo de teclado** - Presiona `Ctrl+B` para acceder a controles en tiempo real
- **Vértices ajustables** - Control de 4-12 puntos para formas más/menos orgánicas
- **Redondez variable** - 20%-80% para controlar suavidad vs. angular
- **Velocidades customizables** - Morphing de 4-20 segundos y colores de 10-30 segundos
- **Regeneración** - Botón para crear formas completamente nuevas

#### 🎨 Sistema de Colores
- **6 colores Catppuccin** - Rosa → Morado → Azul → Verde agua → Verde → Naranja  
- **Transiciones suaves** de 12 segundos entre colores
- **Sincronización perfecta** - Fondo y borde cambian al unísono
- **Efectos de profundidad** - Drop-shadows y hover effects

#### 🖱️ Experiencia Interactiva
- **Auto-rotación inteligente** - Solo cuando no hay interacción del usuario
- **Mouse tracking** - El blob sigue el cursor con interpolación suave
- **Cambio aleatorio** - 15% probabilidad de cambiar imagen con movimiento
- **Scaling dinámico** - Efectos de escala basados en la distancia del cursor

## 📸 Especificaciones para Imágenes del Blob

### 🎨 Imágenes de Perfil (assets/img/profiles/)
Las imágenes para el blob interactivo deben cumplir estas especificaciones:

**Características técnicas:**
- **Formato:** PNG con transparencia
- **Resolución:** 512x512px (mínimo) o 1024x1024px (recomendado)
- **Aspecto:** Cuadrado 1:1
- **Calidad:** Alta resolución para nitidez
- **Fondo:** Transparente o fácilmente removible

**Contenido visual:**
- **Estilo:** Profesional, creativo o casual según el contexto
- **Encuadre:** Busto o retrato (cabeza y hombros)
- **Expresión:** Variada (profesional, sonriente, pensativo, creativo)
- **Iluminación:** Bien iluminado, evitar sombras duras
- **Orientación:** Frontal o 3/4, mirando hacia adelante

**Naming convention:**
- `mg_1.png` a `mg_9.png` - Variaciones de personalidad/contexto
- Usar nombres descriptivos si representan contextos específicos

**Procesamiento:**
1. Redimensionar a 512x512px
2. Remover fondo (usar herramientas como remove.bg)
3. Centrar el sujeto en el frame
4. Optimizar peso del archivo sin perder calidad
5. Asegurar que funcione bien con el recorte orgánico del blob

**Contextos sugeridos:**
- Profesional con traje/camisa
- Casual/relajado
- Creativo/artístico
- Técnico con herramientas
- Empresarial
- Sonriente/amigable
- Serio/enfocado

El sistema rotará automáticamente entre estas imágenes cada 4 segundos y cambiará dinámicamente con la interacción del mouse.
